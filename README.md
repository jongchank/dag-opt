# dag-opt

To build
```
$ cmake .
$ make
```
Usage
```                                                                 
usage: ./opt algorithm dag e1 e2 ... en                                                                                         
  algorithm: [exh|our]                                                                                                         
  dag: DAG type [a|b|c|d]                                                                                                       
  e1 e2 ... en: execution times for each runnable
```
